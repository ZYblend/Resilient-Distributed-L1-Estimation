# Resilient Multi-Camera Perception

## How to run
- run "Run_file.m"
- run "MCCE.slx"
- run "plot_file.m"

## Important functions
- L1_minimization.m
- distributed_L1_minimization.m

Read the comments in the functions for the deduction and example of use. If have any questions, please contact Yu Zheng at yzheng6@fsu.edu.
